(2)

This engine shop manual (ESM) covers the functions necessary to maintain and refurbish the Passport 20 engine in an overhaul environment. The manual is structured to fit the engine’s modular design (modules, assemblies, and individual parts). The instructions are organized to cover two types of maintenance:

  • On-condition maintenance

  • Off-wing level maintenance (shop level work scopes).

(c) Third element (Unit).

The third element (fifth and sixth digits) divides the subsystem into units (modules, assemblies/subassemblies, or piece parts).

(A)

When the assembly/subassembly number is in the second element position and the third element is two zeros (72-XX-00), the procedure is for a module removed from the engine. This work might be partial or complete disassembly and assembly.
(B)

When the assembly/subassembly number is in the second element position and the third element position is utilized (72-XX-YY), the procedure is for a piece part that is removed from the engine. This may include cleaning, inspection, and repair.
(C)

When the assembly/subassembly number is in the third element position and the second element is two zeros (72-00-XX), the procedure is for a condition maintenance. This work might be cleaning or inspection of the module installed on the engine assembly.
(D)

When the unit is equal to 01, 02, or 03 (72-00-0X), this denotes work being performed on the complete engine, as well as for information applicable to removal, installation, or inspection of the module. The disassembly/assembly of modules into/from assemblies or subassemblies is also covered at this level.
(d) Directory numbering.

Within each system/subsystem/unit, information is arranged in individual directories as outlined below:

(A) Description and operation.

This directory includes the description and operation task related to each engine system.
(B) Special procedure.

This directory includes all procedures or references to do a complete, self-contained procedure.
(C) Disassembly/Assembly.

  1. These directories differ depending upon the level of engine maintenance and build status:

    • 72-00-00 has procedures to disassemble/assemble the engine into/from modules and to disassemble/assemble the external piping and hardware on the engine

    • 72-00-0X has procedures to disassemble/assemble the module into/from assemblies and/or subassemblies

    • 72-XX-00 has procedures to disassemble/assemble the assembly/subassembly into/from individual subassemblies or piece parts.

  2. These directories have subfolders with the list of DMs called by the hat data module of the Assembly/Disassembly directory.

(D) Cleaning.

This directory includes the cleaning tasks related to each level of engine maintenance. It concerns the engine, module, and piece part levels.
(E) Inspection/Check.

This directory includes the inspection tasks related to each level of engine maintenance. It concerns the engine, module, and piece part levels.
(F) Storage.

This directory gives instructions for special handling, packaging, storage, transportation, and preservation for a module, assembly, or subassembly. The shelf life is not included in the storage directory.

(G) Testing.

This directory gives the procedures and parameters to make sure that the repaired or overhauled engines meet operational and performance standards. These procedures are written to the availability of complete shop/test facilities. If complete test facilities are not available and modular maintenance is to be done with the use of procedures contained in this manual, the operator should use test recommendations found in the aircraft maintenance manual for engine integrity checks.
(e) S1000D Data Module Functions.
(A) Content.

The manual provides engine maintenance data produced in a modular form called a DM. Each DM contains the following information:

  • An identification and status section with all management information:

    • Identification data (including data module code (DMC) , title, issue number, issue date)

    • Status data (including security classification, responsible partner company and originator, applicability, reason for update).

  • A content section including:

    • The required conditions to be met, if applicable

    • Support equipment required for the procedure

    • A list of required supplies (consumables, solutions, and expendables; test specimens, as applicable)

    • Identification of spares

    • Safety conditions.

  • Each DM represents a unique information set identified by a DMC.

(B) General data module codification.

  1. The DMC is the standardized and structured identifier of a DM. It is located in the identification section of the DM.

  2. DMs are coded using characters as follows:

    • PASSPORT20-AX-YY-YY-YY-NNA-YYYA-E.

      Where

      • "PASSPORT20" represents the model identification (MI) code allocated to the Passport 20 project. It identifies the product to which the data applies and is the point of reference for all applicability information.

      • "AX" represents the system difference code (SDC) . It identifies alternative versions of the system and subsystem/sub-subsystem identified by the SNS without affecting the type, model, or variant identity. "00" will be used for generic DMs such as illustrated tools and equipment manual (ITEM) sheets.

      • The following "YY-YY-YY" is the SNS. This system is designed to provide standardization in the arrangement or addressing to the product. For the ESM, it is usually the engine breakdown.

        Note:  The third part of service bulletins (SB) SNS contains four digits. These last four digits are the SB number.

      • The following "NN" is the disassembly code (DC) . The DC for the Passport 20 ESM project starts from 02. In some circumstances, the DC can be "00" for tasks performed on "virtual" set of components. It identifies the breakdown condition of an assembly to which maintenance information applies. It is also used for sequential numbering of DM when more than one DM is used for the same SNS.

      • The following "A" represents the disassembly code variant (DCV) . It designates alternative items of equipment or components differing slightly in design but not enough to warrant the change of the SDC. It includes the identification of procedures written specifically for the left or right engine.

      • The following "YYY" is the information code or "info code." It identifies the type of information within a DM.

      • The "A" after the information code is the information code variant (ICV) . It identifies any variation in the activity defined by the information code. Generally, it is used to distinguish various methods existing for a given procedure.

      • The last "E" is the item location code (ILC) . It identifies the situation to which the information is applicable, where the maintenance task will be carried out.

        • "A" value is related to items installed on the aircraft (used for line maintenance).

        • "C" value is related to items installed on the engine at overhaul or installed on the engine in the shop.

        • "D" value is related to items installed on the engine (line maintenance).

(3) Assembly/Disassembly.
(a) Hat data module.
(A)

A hat data module gives instruction to assemble or disassemble the engine assembly, the major modules and the minor modules. This hat data module calls the DMs in the order that allow an efficient overhaul.

(B)

To keep the length and complexity of the engine assembly level to a minimum, procedures are divided into multiple hat data modules:

  • Assembly/Disassembly of the engine assembly,

  • Assembly/Disassembly of the core module assembly,

  • Assembly/Disassembly of the low pressure turbine module assembly,

  • Assembly/Disassembly of the accessory gearbox module assembly hardware,

  • Assembly/Disassembly of the aft fan case assembly hardware,

  • Installation/Removal of the booster assembly,

  • Assembly/Disassembly of the No. 1 and No. 2 bearing assembly,

  • Assembly/Disassembly of the fan hub frame assembly,

  • Assembly/Disassembly of the high pressure compressor rotor assembly,

  • Assembly/Disassembly of the high pressure compressor stator assembly,

  • Assembly/Disassembly of the combustor diffuser nozzle assembly,

  • Assembly/Disassembly of the high pressure turbine stator assembly,

  • Assembly/Disassembly of the high pressure turbine rotor assembly,

  • Assembly/Disassembly of the turbine center frame,

  • Assembly/Disassembly of the low pressure turbine rotor/stator assembly,

  • Assembly/Disassembly of the inlet gearbox assembly,

  • Assembly/Disassembly of the accessory gearbox assembly.
(b) Modular installation/removal.

The removal and installation of a module is described in the assembly/disassembly of the upper module.

  • Example: The removal of the fan hub frame (FHF) module is given in the disassembly of the engine assembly.

(c) Torque values.

The torque used in the procedures is given with a 8% maximum tolerance and is rounded inside the tolerance interval. This tolerance takes into account the tolerance of the tools. The operator must use the median value and no other tolerance has to be added.

  • Example: "Torque to 106 to 124 lb in. (12 to 14 Nm)".

    The median value to be used is 115 lb in. (13.0 Nm). The 8% tolerance gives an interval from 106 lb in. (12.0 Nm) to 124 lb in. (14.0 Nm). The rounding used in the procedure gives torque value inside the interval.

(d) Storage.

This directory gives instructions for special handling, packaging, storage, transportation, and preservation for a module, assembly, or subassembly. The storage, transportation, and preservation of the engine is in the Line Maintenance Manual (GEK 112062).

(5) Inspection.
Note:  These procedures are written with the understanding that all engine parts/assemblies have been cleaned prior to inspection or are clean enough as they are to ensure that an adequate assessment of part condition can be made and in turn a subsequent repair of all findings.
(a)

Inspection DMs provide information concerning part or assembly areas that require inspection. Each DM specifies methods and, if technically necessary, the equipment required. It also identifies critical stress areas where findings are likely to occur or are unacceptable, the nature and maximum allowable limits of findings in all areas and, if applicable, repairable limits.

  • Separate levels of serviceability are given for different degrees of maintenance. Engine areas exposed during removal of a module require checks that may be different from an individual part inspection. The modular inspections are given in 72-00-XX and the piece part inspections are given in 72-XX-YY.

  • Inspect the part with the appropriate method to determine what finding is present. If a finding is located, check the maximum serviceable limits and maximum repairable ones. If the finding is above the maximum serviceable limit and if no repair is available, replace the part. If there is no available criteria for this finding within the manual, contact GE or replace the part.

Note:  For planning purposes, some repair method lines may not contain a repair reference or appear at all. This is to indicate that a repair is not currently available but will be developed by GE as the need arises.

(8) Exposed hardware inspection requirements.
(a)

An individually licensed technician or a licensed repair station (if there is no individually licensed technician), is required to do a visual inspection of all engine parts that can be seen before the engine is assembled again. This includes the items that follow:

  • All parts that are removed from the engine that will be installed again

  • All parts that are not removed from the engine that can now be seen.
(b)

The inspection of these parts must be thorough and the parts must meet the serviceability limits given in the inspection limits that are applicable for the level of maintenance.
(c)

A part does not have to be further disassembled if the area that can be seen is within the serviceable limits.

8. Torque definition.
(1)

The terms used in the ESM are defined as follows:

  • Net torque: Gross torque minus run-on torque.

  • Gross torque: The actual torque applied by the wrench, indicated by the dial or scale on the wrench or the wrench setting.

  • Run-on torque: The torque required to turn the nut or bolt before it is completely seated so that it performs a tightening action.
(2)

Additional information is contained in the Standard Practices Manual (GEK 9250). Refer to SPM TASK 70-51-00-400-004 (Tightening Practices and Torque Values) .

9. Engine directional references.

Clockwise and counterclockwise clock position and other directional references apply to the engine in a horizontal position, viewed from the rear and with the accessories section at the bottom. When components or struts are numbered in circumferential direction, the No. 1 position is at 12:00 o'clock or immediately clockwise from 12:00 o'clock. The remaining positions increase arithmetically in a clockwise direction.
10. Flange identification

The external flanges of the engine have been assigned letter designations. The letter designation will be used for flange identification wherever it is necessary to be explicit about flange location, such as positioning of brackets, clamps, bolts, etc.
11. Supplemental information.

The limits (dimensional) or values (torque), etc., are expressed in the English and metric systems of measurement. The English system is primary followed by the metric conversion.

2. Purpose

The illustrated parts data (IPD) was prepared as specified by S1000D 4.1. It is intended as a companion document of the Engine Shop Manual, not a substitute for, the associated Engine Shop Manual which shall be considered the authority on disassembly/assembly procedures.

The IPD lists and illustrates replaceable assemblies and detail parts removed in the Engine Shop Manual, assists in provisioning and requisitioning.

This Engine illustrated parts data (EIPD) is arranged in one section according to the type design structure being:

  • ENGINE (CHAPTERS 72 THRU 75 and 77 THRU 79)

The section consists of:

  • DETAILED PARTS LIST
3. Detailed parts list - Arrangement

The detailed parts list (DPL) shows the engine or assemblies divided in sections, identified by catalog sequence number (CSN) , which have a relation to Engine Shop Manual.

The indenture system used in the DPL of this catalog shows the relationship of one part to other parts in disassembly order where applicable.

For a specified item, the indenture system shows the primary or general group in the left position. The sequence continues down and to the right until the last detail part is shown, as follows:

5. Detail parts list coding

The model apply code is shown in the usable on code assy column of the detailed parts list.

  • When all information for a CSN applies to all models, no model apply code is shown.

  • When all information for a CSN applies to a specific model(s), the model apply code (or model value) is at the first indenture on the last line of the nomenclature column.

  • When the model apply code (or model value) is at the fourth indenture, only the line above it is applicable to that model(s). All other information applies to all models.

  • A combination of 2 and 3 is possible.

7. Usage on code assembly

Function Numbering.

Before applying the "usage code", make sure that the parts are coded for use on the same engine model (reference Model Apply in nomenclature column). If the part is not coded for the same engine model, the usage code does not apply.

Within a Figure, a part number identified by a "usage code" (a single alpha designation) denotes that the affected part MUST BE USED only with all next higher assembly identified with the same single alpha designation. It also denotes that the affected part can be used with all other non-coded parts (no alpha designation).

When a combination of alphas are used (such as "abd"), the affected parts may be used with all other non-coded parts and all next higher assembly coded "a" or "b" or "d" or any combination thereof (such as " ab" or "ad" or "bd" or "abd"). A part coded "abd" cannot be used with any coded part number which does not contain "a" or "b" or "d" in the alpha combination.

USAGE ON CODE ASSEMBLY will identify affected parts within a Chapter/Section/Unit/Figure only. An alpha code may be repeated from figure to figure, but is only to be used for identification of affected parts within its related figure (i.e. USAGE CODE "a" for 72-54-00 figure 1 does not relate to USAGE CODE "a" for 72-54-00 figure 5).

Note:  If interchangeability of assemblies, sub-assemblies and detail parts is affected between Chapter/Section/Unit/Installation (or between figures thereof), "use with" notes will be used in the nomenclature column for the affected part number(s).

Fully interchangeable parts do not require USAGE CODE designation, or notes, within the assembly level they support.
8. How to use the EIPD
(1)

To identify an item when only the part number is known.

(a)

Find the part number in the Numerical Index and note the Chapter/Section/Subject, Figure and Item Number.
(b)

Turn to this Figure/Item Number in the DPL.

(c)

For description of desired part, refer to Nomenclature.
(d)

For appearance of desired part, locate Item Number on the illustration.
(2)

To identify an item when the part number is not known.

(a)

Turn to the Chapter Table of Contents for the Chapter/Section/Subject in which the major assembly appears and find the Figure Number.
(b)

Turn to the Figure where this major assembly is illustrated.
(c)

Note the Item Number on the illustration.
(d)

Locate the Item Number in the Detailed Parts List which will show the part number and Nomenclature.

0

Part delivered in finished state.

1 thru 5

Parts finished or semi-finished with material allowance from first to fifth repair.

6

Permanently-assembled unit delivered in semi-finished state. Completed on NHA. To determine the necessary operations to assemble these parts, refer to the applicable shop manual procedures.

8

9
